## **About the app -**

- Landing page contains list of 5 books.
- Pagination is added to render list of next 5 books.
- By clicking card user can navigate to book detail page.
- Clicking back button will leave user on landing page again.

## **Tech Stack -**

- React JS
- Redux-toolkit
- Tailwind CSS
